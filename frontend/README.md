# E-Commerce Product Page with React

This is a feature-rich e-commerce web application built with React, providing users with a seamless product browsing and shopping cart experience. The project demonstrates modern React development practices, including Context API for state management, responsive design, and reusable components.

---

## Features

- **Dynamic Product Listing**: View a variety of products with details such as name, price, description, and stock availability.
- **Product Details Page**: Detailed information for each product, including an image and additional metadata.
- **Add to Cart Functionality**: Users can add products to their cart, adjust quantities, and view their total.
- **Cart Page**: Displays selected products, total cost, and allows users to remove items.
- **Responsive Design**: Optimized for all screen sizes, from desktops to mobile devices.
- **Modern Styling**: Dark theme with vibrant buttons and user-friendly layout.

---

## Technologies Used

- **Frontend**:
  - [React](https://reactjs.org/) - Component-based JavaScript library for building user interfaces.
  - [React Router](https://reactrouter.com/) - For handling client-side routing.
  - CSS for styling, including responsive design.

## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/aclarke500/react-rag-store.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ecommerce-react-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

The app will run on [http://localhost:3000](http://localhost:3000).
